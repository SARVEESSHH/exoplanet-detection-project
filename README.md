# 🌌 Exoplanet Detection Project

This project uses NASA Kepler Space Telescope data to detect potential exoplanets using the transit method.

## 🔭 Objective
To identify periodic brightness dips in stars that may indicate orbiting exoplanets.

## 🧪 Methods
- Light curve analysis using Python
- Data cleaning and normalization
- Periodogram analysis
- Phase folding technique

## 📊 Results
- Detected periodic signals in selected star systems
- Generated light curve visualizations
- Estimated possible planetary signals

## 🛠️ Tools
- Python
- Lightkurve
- Matplotlib
- NASA Kepler dataset

## 🌠 Key Insight
Astronomical data analysis can reveal hidden planetary systems light-years away.

# 🌌 Exoplanet Detection Mission Dashboard

A research-style astronomy project that analyzes NASA Kepler telescope data using Python and Lightkurve.

## Objectives
- Detect periodic signals in stellar brightness data
- Generate light curves and periodograms
- Evaluate possible exoplanet transit signatures

## Results
A periodic signal of 0.523 days (12.6 hours) was identified. Phase-folded analysis did not reveal a definitive transit signature, suggesting additional validation is required.

## Tools
- Python
- Lightkurve
- Matplotlib
- NASA Kepler Data
