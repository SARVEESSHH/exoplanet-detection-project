🌌 Exoplanet Detection Using NASA Kepler Data

This project analyses real NASA telescope data to detect possible exoplanets using the transit method.

🔭 What I Did

  • Collected light curve data from NASA Kepler mission
  • Cleaned and processed time-series data using Python
  • Detected periodic brightness dips
  • Estimated possible exoplanet signals
